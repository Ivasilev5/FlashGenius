# 📚 FlashCard App

Мобильное приложение для изучения флеш-карточек с поддержкой ИИ-агента.
Аналог Anki / Quizlet с алгоритмом интервального повторения (SRS).

## Возможности

- Создание колод и карточек (вопрос / ответ)
- Умный алгоритм повторения SM-2 (Spaced Repetition)
- Оценка сложности после каждой карточки (Снова / Сложно / Хорошо / Легко)
- ИИ-агент: генерация карточек по теме
- ИИ-агент: создание карточек из PDF файла
- Статистика прогресса
